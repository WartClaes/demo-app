# Inimco project

## Getting started

The repository exists of 2 separate parts.

**Backend**: backend build with Express
**Frontend**: frontend build with Angular

### Backend

```
$ cd backend
$ npm install
$ npm run dev
```

### Frontend

```
$ cd frontend
$ npm install
$ npm start
```

## Testing

The backend repository includes tests for the utility functions

```
$ cd backend
$ npm run test
```
